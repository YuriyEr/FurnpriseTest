Нужно составить блок-схему универсального алгоритма по расчёту материала, необходимого для того,
чтобы им замостить некую площадь.
Например есть стена/пол (любая плоская поверхность) и нужно посчитать кол-во досок/плиток/листов.
Кол-во материала должно быть минимизировано, т.е. с использованием обрезков, но кол-во обрезков должно быть,
так же, минимизировано. Размер обрезка тоже имеет ограничение.

Что дано:
Параметры плоской поверхности: длина, ширина
Параметры материала: длина, ширина
Минимальные параметры обрезка: длина, ширина
Интервал, через который нужно выкладывать материал: по длине, по ширине. (т.е. зазор, но он может быть задан равным нулю)

Результат задания:
блок-схема алгоритма вычисления с небольшим описанием для пояснения
Формат - любой (хоть карандашом на бумаге), Enterprice Architect - как плюс.
